# Alpine Runner
A dinosaur game clone with a different theme with a mountain goat as the dinosaur, a mountain as the cactus, and an eagle as the teradactal.

# What Is Needed To Run The Game
This Java Swing game requires the JDK (download from oracle.com/java). In a terminal, go to src/, run javac *.java then java Main. For VS Code users, install the Java Extension Pack and use the Run button.
